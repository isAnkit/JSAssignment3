4. Find the sum of the digits of a given number


  Write a program that takes a number from the user and get the sum of the digits present in the number.
  
    >> Answer

    

        
        const Number_Sum = (N) => 
        {
            let sum = 0;
            while (N) {
                x = N % 10;
                sum += x;
                N = (N - x) / 10;
            }
            return sum;
        };
